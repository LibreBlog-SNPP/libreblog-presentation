# LibreBlog - Presentación del Proyecto

Presentación académica desarrollada con Reveal.js para el proyecto "Sistema Web para Creación y Administración de Blogs Personales - LibreBlog".

##  Contenido

La presentación incluye 14 diapositivas que cubren:

- **Portada institucional** - Centro Tecnológico de Formación Profesional Paraguay – Japón
- **Introducción** - Contexto del problema y propuesta de solución
- **Planteamiento del problema** - Desafíos técnicos y pregunta de investigación
- **Objetivos** - Objetivos generales y específicos del proyecto
- **Justificación** - Justificación académica y social
- **Marco Teórico** - Fundamentos de arquitectura serverless y seguridad
- **Marco Metodológico** - Tipo de investigación y técnicas utilizadas
- **Arquitectura Técnica** - Stack tecnológico (Next.js, React, Supabase)
- **Demostración** - Enlaces a la aplicación y repositorio
- **Resultados** - Métricas de rendimiento y usabilidad
- **Conclusiones** - Aportes y trabajo futuro

##  Cómo usar

### Ver localmente
1. Abre el archivo `index.html` en tu navegador

### Navegación
- **Flechas ←/→** - Navegar entre diapositivas
- **Espacio** - Siguiente diapositiva
- **Esc** - Vista general
- **F** - Pantalla completa
- **S** - Modo presentador

##  Tecnologías

- Reveal.js 4.3.1
- CSS Grid y Flexbox
- Times New Roman (tipografía académica)
