SISTEMA WEB PARA CREACIÓN Y ADMINISTRACIÓN DE BLOGS PERSONALES

LibreBlog Plataforma de Blogging de Código Abierto

Introducción

Contexto:

  • Plataformas cerradas → pérdida de control de datos para los creadores
  • Infraestructura propia → coste y complejidad

Problema:

  • Falta de alternativas seguras y económicas para publicar contenido

Propuesta:

  • LibreBlog: Open Source, control de datos a nivel de instancia, coste operativo cero en entorno free-tier

Planteamiento del problema

Desafío técnico:

  • Implementar seguridad avanzada y tiempo real sin servidores propios

Pregunta de investigación:

  • ¿Es viable una plataforma segura y reactiva usando únicamente servicios gratuitos (free-tier)?

Oportunidad tecnológica:

  • Madurez del ecosistema Serverless (Next.js, React, Supabase, Vercel)

Objetivo

Objetivo general:

  • Desarrollar y validar una plataforma web de blogging utilizando tecnologías modernas y gratuitas.

Objetivos específicos:

  • Diseñar una arquitectura serverless utilizando Next.js y React.
  • Implementar un sistema de autenticación y autorización seguro.
  • Aplicar políticas de seguridad a nivel de base de datos mediante Row Level Security (RLS).
  • Evaluar el rendimiento y la usabilidad de la plataforma desarrollada.

Justificación

Justificación académica:

  • Caso de estudio replicable para proyectos de desarrollo web moderno.
  • Aplicación práctica de conceptos de arquitectura, seguridad y rendimiento.

Justificación social:

  • Promoción del software libre y el acceso a plataformas web seguras.
  • Reducción de barreras económicas para la publicación de contenido digital.

Marco Teórico

Arquitectura Serverless:

  • Eliminación de servidores dedicados
  • Escalabilidad gestionada por el proveedor

Frameworks Web Modernos:

  • Next.js y React como estándares actuales

Seguridad de la Información:

  • Autenticación basada en JWT
  • Control de acceso mediante Row Level Security (RLS)

Marco Metodológico

Tipo de investigación:

  • Aplicada – Tecnológica

Diseño de investigación:

  • No experimental – Descriptivo

Enfoque metodológico:
